記事の追加方法

サイト運用 / 2026-07-08

この技術メモは、ビルドツールなしの素の HTML/CSS/JS だけで動いている。記事の追加は次の2ステップで完了する。

手順1: テンプレートをコピーして記事 HTML を作る

tech/articles/template.html を同じフォルダにコピーし、ファイル名を記事の slug（英数字とハイフン）にする。例: my-new-article.html。中の <title>・<h1>・.tech-meta・本文を書き換える。

手順2: articles.js に1行追加する

tech/articles.js の配列の先頭に、次の形のエントリを追加する。

{
  slug: "my-new-article",        // ファイル名と一致させる
  title: "記事のタイトル",
  category: "カテゴリ名",         // 既存と同名なら同じグループに入る
  date: "2026-07-08",
  description: "一覧に表示される説明文。"
},

slug がファイル名と一致していれば、サイドバーと一覧ページに自動で表示される。

公開する

変更を GitHub の main ブランチに push すれば、GitHub Pages が自動で makeinufilm.com に反映する。ローカル確認はブラウザで tech/index.html を直接開くだけでよい（サーバ不要）。

書ける要素

template.html のコメントに書き方の例がある。見出しは <h2>/<h3>、コードは <pre><code>、画像は img/pic/ に置いて <img>、表は <div class="table-wrap"><table> で囲むとスマホで横スクロールになる。
