makeinufilm.com 技術メモ

このサイトの構成メモ

サイト運用 / 2026-07-08

makeinufilm.com は GitHub Pages（リポジトリ inuFilm/pr_webpage の main ブランチ直下）で配信している純静的サイト。ビルド工程はない。

トップページ

index.html（日本語）と en.html（英語）は Bootstrap 3 ベースのシングルページ。jQuery + onePageNav でセクションスクロールを制御している。

技術メモセクション

tech/ 配下はトップページとは独立した作りで、Bootstrap にも jQuery にも依存しない。記事メタデータは tech/articles.js の配列1つで管理し、tech/tech-nav.js がサイドバーと記事一覧を描画する。スタイルは tech/tech.css の1枚のみ。

  • tech/index.html — 記事一覧
  • tech/articles/*.html — 各記事（template.html をコピーして作る）
  • .nojekyll — GitHub Pages の Jekyll 処理を無効化

デプロイ

main ブランチに push するだけ。カスタムドメインは CNAME ファイル（makeinufilm.com）で設定している。
